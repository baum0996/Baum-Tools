# Changelog

All notable changes to BAUM TOOLS V1 will be documented in this file.

## [1.0.0] - 2026-06-26

### Added

- Initial modular terminal application structure.
- Rich-based category navigation.
- Settings system backed by `settings.json`.
- Native Windows folder and file selection.
- Minecraft category.
- Texture Pack Scanner for folders and ZIP archives.
- Windows category.
- Startup Analyzer for read-only startup entry inspection.
- System Info Dashboard for read-only Windows system information.
- Executable and script-like file type detection.
- Professional `start.bat` launcher with virtual environment setup.
- Automatic dependency installation from `requirements.txt`.
- GitHub-ready repository files.
